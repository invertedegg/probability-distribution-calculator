Probability Distribution Calculator
Author: invertedegg (www.github.com/invertedegg)
-------------------------------------------

DESCRIPTION:
This GitHub repository contains a project with the goal of providing fast, easy computation
of probability based on user input. Users are prompted to select a probability distribution
to solve for, then are asked to enter relavent data. 

EXECUTION:
- Navigate to the folder containing the file "ProbabilityCalculator" in the terminal
- Execute the command "./ProbabilityCalculator"
- Follow the terminal prompts
